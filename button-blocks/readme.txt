== Description ==

The **Button Blocks** plugin allows you to easily add customizable and responsive buttons to your WordPress website. Perfect for both beginners and developers, this plugin provides a simple yet powerful block for the WordPress editor (Gutenberg). Customize button alignment, text, URLs, and more, while enjoying full control over button styling and functionality.

== Key Features: ==

- **Customizable Button Alignment:** Easily adjust your button's position to the left, right, or center for a seamless integration with your content layout.
- **Editable Button Text:** Modify the button label to fit your call-to-action needs (e.g., "Learn More," "Shop Now," or "Contact Us").
- **URL Support:** Attach custom URLs to buttons, directing users to specific pages, forms, or external websites.
- **Target Attribute:** Decide whether the button link opens in the same tab or a new one for better navigation control.
- **Advanced Styling Options::** Tailor the button’s design to match your site’s theme by adjusting colors, sizes, borders, and more.
- **Responsive Design:** Ensure your buttons are mobile-friendly with automatic adjustments for different screen sizes (desktop, tablet, and mobile).
- **User-Friendly Interface:** Seamlessly integrate with the WordPress block editor (Gutenberg), making button customization intuitive without the need for coding.
- **Extensive Color Options:** Enjoy support for 50 default colors and 100 gradient color options, offering a wide range of possibilities for your button's appearance.
- **Custom Gradient Support:** Take full control of your button's background with the ability to apply custom gradient colors, allowing for unique and dynamic button designs that stand out.

== Installation ==

There is no prerequisites for installing **Button Blocks**. You can follow the steps below to complete the installation.

= Automatic Installation =
1.  Go to `Plugins > Add New` screen from your WordPress website dashboard.
2.  Search for `Button Blocks`.
3.  Install and activate the plugin, that's it!
